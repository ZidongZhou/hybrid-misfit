## Full reproduction command

The raw data are not redistributed. Supply the verified SWAA archive with:

```bash
python code/run_all.py --data-zip /path/to/WFH_Code_and_Data_May2020_to_May2026.zip
```

The core pipeline checks the SHA256 value in `DATA_MANIFEST.md`, deletes generated outputs, and rebuilds the analysis tables, source CSV files, 600 dpi PNG figures, MDPI Word files, formatted supplementary workbook, output manifests, detailed reproducibility log, and automated tests. The standard command is a complete submission rebuild and generates the formatted supplementary workbook with `artifact_tool`. If the formatter is unavailable, `--skip-workbook` may be used for a scientific-output-only run; such a run is not the final submission build.

## Main reporting decisions

- The fully adjusted association model uses 15,355 complete cases from September–December 2025, equal to 41.0% of the 37,434 planned-reference observations from July 2025 through February 2026.
- The primary reference is employer-planned WFH; combined and current-WFH references are separate sensitivity definitions.
- The main complex comparison is one planned-only **reduced-feature random forest**. Current WFH is excluded from all models in Supplementary Table S30.
- At the evaluated 10% capacity, the **under-remote directional rule** is the prespecified transparent benchmark in Supplementary Table S41.
- The five-category outcome sensitivity is described as a satisfaction-score analysis, not as an ordered model.
- Response-surface variables use the same 2.5-day center, and sparsely supported prediction regions are masked.

## Data scope and access

The analysis uses `WFHdata_May26.csv` from the provider’s earnings-restricted June 2026 SWAA package. The alternative no-earnings-requirement file is not used. Researchers can access anonymized SWAA microdata after account creation at `https://wfhresearch.com/data/`, subject to provider terms. See `DATA_MANIFEST.md` for the archive size, internal file names, release definition, and SHA256 value.
